# 📚 Study Group Finder

A full-stack web application that helps students discover, create, and join study groups based on shared subjects, skills, and availability.

Built with React.js and Firebase, this project is designed to foster collaboration, boost productivity, and encourage peer-to-peer learning in academic environments.

## ✨ Features

- 👤 User Authentication (Login, Signup)
- 🧾 Create and Join Study Groups
- 🔍 Filter Groups by Subject, Time, or Skills
- 🗂 User Profiles with Interests & Availability
- 💬 Messaging (Realtime Chat — planned)
- 🧠 Smart Matching (based on skills — planned)
